# Shadcn Vue Hooks

A comprehensive collection of reusable Vue composables designed specifically for the shadcn-vue ecosystem. This project provides production-ready hooks that solve common development challenges and enhance your Vue applications with powerful, type-safe utilities.

## What is shadcn-vue-hooks?

**shadcn-vue-hooks** is a curated library of Vue 3 composables that extend the functionality of shadcn-vue components. Built on top of modern Vue 3 Composition API and TypeScript, these hooks provide reusable logic for common patterns in component development, state management, and user interactions.

## Installation & Usage

### Prerequisites

Ensure you have a working Vue 3 project with shadcn-vue installed:

```bash
npx shadcn-vue@latest init
```

### Installing Hooks

Add our custom registry to your `components.json`:

```json
{
  "registries": {
    "@shadcn-vue-hooks": "https://shadcn-vue-hooks.vercel.app/r/{name}.json"
  }
}
```

Then install individual hooks:

```bash
# Install the context management hook
npx shadcn-vue@latest add @shadcn-vue-hooks/createContext

# Install other hooks as they become available
npx shadcn-vue@latest add @shadcn-vue-hooks/useForwardPropsEmits
```

### Basic Usage

```vue
<template>
  <div>
    <button @click="toggleTheme">
      Current Theme: {{ config.theme }}
    </button>
  </div>
</template>

<script setup lang="ts">
import { injectConfigContext } from '@/composables/createContext'

const config = injectConfigContext()

  // Implementation logic here
}
</script>
```
