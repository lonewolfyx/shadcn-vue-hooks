## Prerequisites

### 1. Initialize shadcn-vue

Before installing shadcn-vue-hooks, you need to have a working shadcn-vue project. If you haven't set up shadcn-vue yet, follow these steps:

Visit the official [shadcn-vue documentation](https://shadcn-vue.com) for detailed installation instructions.

```bash
# Initialize shadcn-vue in your Vue project
npx shadcn-vue@latest init
```

This will create a `components.json` configuration file in your project root and set up the necessary dependencies.

### 2. Project Requirements

Ensure your project meets these requirements:

- **Vue 3.5+** with Composition API support
- **TypeScript** enabled (recommended)
- **Tailwind CSS** configured

## Configuration

### Configure components.json

Add the shadcn-vue-hooks registry to your `components.json` file:

```json
{
  "$schema": "https://shadcn-vue.com/schema.json",
  "style": "new-york",
  "typescript": true,
  "tailwind": {
    "config": "tailwind.config.js",
    "css": "src/assets/css/main.css",
    "baseColor": "neutral",
    "cssVariables": true,
    "prefix": ""
  },
  "rsc": false,
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils",
    "ui": "@/components/ui",
    "lib": "@/lib",
    "composables": "@/composables"
  },
  "registries": {
    "@shadcn-vue-hooks": "https://shadcn-vue-hooks.vercel.app/r/{name}.json"
  }
}
```



## Installing Composables

### Install Available Hooks

Use the shadcn-vue CLI to install hooks:

```bash
npx shadcn-vue@latest add @shadcn-vue-hooks/createContext
```


## Troubleshooting

### Common Issues

#### 1. Registry Not Found
```
Error: Registry not found at https://shadcn-vue-hooks.vercel.app/r/{name}.json
```

**Solution**: Ensure your `components.json` has the correct registry URL and your internet connection is working.
